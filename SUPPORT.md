# Support

Use GitHub Issues for reproducible bugs in the plugin, Skills, schemas, docs, or validation scripts.

For component package help, include:

- Widget Workshop version.
- The failing `manifest.json` and `scripts/settings.json` if present.
- The output from `scripts/Test-ComponentPackage.ps1`.
- The exact Host API error code, such as `PermissionDenied`, `MethodNotFound`, `InvalidArguments`, or `FetchBlocked`.
