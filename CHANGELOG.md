# Changelog

All notable changes to Widget Workshop Dev are documented here.

## Unreleased

- Replaced the plugin Skill surface with `widget-workshop-workflow`, `widget-workshop-api`, `widget-workshop-code-review`, and `widget-workshop-dev`.
- Renamed the component workflow agent wrapper so `widget-workshop-dev` now refers only to component code authoring.

## 0.1.0 - 2026-05-23

- Initial standalone release of the Widget Workshop developer toolkit.
- Added installable `widget-workshop` plugin with orchestrator agent and six focused Skills.
- Added bilingual component developer docs, Host API contracts, manifest/settings schemas, and a minimal example component.
- Added repository, plugin, component, and host-contract validation scripts.
- Added GitHub Actions validation and release-policy workflows.
